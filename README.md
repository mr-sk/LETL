LETL
====

Luigi ETL

Installation
```
git clone https://github.com/spotify/luigi
cd luigi
python setup.py install 

pip install tornado

easy_install mechanize
emacs client.cfg
[hadoop]
version: cdh4
streaming-jar: /usr/local/Cellar/hadoop/2.3.0/libexec/share/hadoop/tools/lib/hadoop-streaming-2.3.0.jar
```
